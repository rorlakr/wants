> **주요기능**

: 회원은 두가지 기능을 사용할 수 있다. 실제로 구직도 할 수 있고, 구인도 할 수 있다. 관련서적도 등록할 수 있게...

이 서비스의 구인구직 데이터는 `RORLAB 주간뉴스`에 광고한다.
